## 简历描述草稿

项目名称：智能 PDF 知识库问答助手

项目描述：

基于 Python、Streamlit、pypdf、sentence-transformers 和 DeepSeek API 开发的 PDF 问答应用，实现 PDF 文本提取、文本分块、Embedding 语义检索、来源标注、聊天记录和 API 异常处理，完成轻量级 RAG 问答原型。

后续升级后可补充：

引入 Streamlit 缓存机制优化 PDF 解析效率，并使用 Embedding 实现语义检索；后续计划接入向量数据库，实现向量持久化和多文档检索。
